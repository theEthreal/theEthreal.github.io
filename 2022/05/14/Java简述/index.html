java简述

发表于2022-05-14|更新于2022-05-20|java
|阅读量:

C and C++

  • 1982年C++诞生
  • 面向对象
  • 图形、游戏领域

Java

  • Java 2标准版（J2SE）：桌面端
  • java 2移动版（J2ME）：手机端
  • Java 2企业版（J2EE）：服务器端

三高问题：高性能，高可用，高并发

基于Java开发的系统、平台、工具

  • 构建工具：Ant、Maven、Jekins
  • 应用服务器：Tomcat、Jetty、Jboss、Websphere、weblogic
  • Web开发：Struts、Spring、myBatis
  • 开发工具：Eclipse、Idea、Netbean、Intellij
  • 2006年基于java分布式管理出现了Hadoop-大数据领域
  • 2008年Android的发展和流行

Java的特性和优势

  • 简单性
  • 面向对象
  • 可移植性
  • 高性能
  • 分布式
  • 动态性——反射机制
  • 多线程
  • 安全性
  • 健壮性

Java成功的原因

顺应时代的发展，贴合了互联网时代来临的需求，自身足够优秀
